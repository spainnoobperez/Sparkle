Sparkle

Звук

    1. Скомпилируйте и установите драйвер из архива pcm_sparkle.tar.gz:
# в контейнере
make
sudo make install
Кроме компилятора, наверняка понадобится установить пакет с заголовками alsa. В debian он называется libasound2-dev, в fedora - alsa-lib-devel.
    2. Переименуйте файл 1.asoundrc в .asoundrc и поместите в домашнюю директорию пользователя (~/).
    3. Для проверки запустите mplayer или mpv как-то так:
mplayer -vo x11 -ao alsa -lavdopts threads=4 anime.mkv
Есть вероятность (если не установлено pkg-config, то стопроцентная), что драйвер установится не в ту директорию. Тогда проигрыватель выбьет ошибку и покажет, в какой директории должен быть драйвер. Скопируйте его туда вручную.
