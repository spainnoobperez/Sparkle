Далее описывается много всякой сложной хрени. Это нужно для тех, кто хочет всё понять и сделать вручную. В принципе, понимать и делать всю эту хрень не обязательно. Вот самая простая инструкция: 1. Установить и запустить apk. 2. Смонтировать дистрибутив в Linux Deploy. Установить в нем Xwayland. Убедиться, что в домашней директории ([file]~/[/file]) пользователя есть файл [file].xinitrc[/file], в котором указано, какое запускать окружение (или хоть xterm). 3. Скачать скрипт [file]user.sh[/file], изменить в нем путь к точке монтирования дистрибутива и имя пользователя. 4. Забросить скрипт на устройство и запустить вот так: [code] # в андроиде ./user.sh start [/code] Если всё заработает, можно установить скрипт вот так: [code] # в андроиде ./user.sh install [/code] тогда он будет выполняться автоматически при старте sparkle. А если не заработает, покажите вывод скрипта в теме Linux Deploy и старик Батуев его исправит. Если хотите, можно добавить в скрипт несколько строчек (mount-ов) и тогда Linux Deploy будет не нужен.
